== Description ==

WCM Language Switcher adds a button to the admin toolbar that allows users to seamlessly switch between available languages.

 

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

It scans the language directory to build the list of available languages. To add a language, simple <a href=\"http://codex.wordpress.org/WordPress\_in\_Your_Language\" title=\"WordPress In Your Language\">download the appropriate mo file</a> and add to your WordPress language folder.

 

This plug-in was originally built as a means of plug-in and theme developers to test translations of their plug-ins or themes.
